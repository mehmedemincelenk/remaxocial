# Using Remotion Bits

- **`remotion-bits`** is a collection of high-quality, composable components (bits) for programmatic video.
- **MCP Integration:** Use the `remotion-bits` MCP server to find and fetch components.
- **Key Categories:**
  - **Text:** Specialized reveals, blur effects, and staggered animations.
  - **Visuals:** Confetti, rain, and particle systems.
  - **3D:** Ready-made React Three Fiber scenes.
- **Workflow:**
  1. Ask the agent to "Find a bit for [feature]".
  2. Use the fetched code directly in your templates.
  3. Ensure all imported bits follow the project's "Diamond Standard" styling.
